Pneumono Detect

  • Tech Stack: CNN, ResNet50, Keras, Hugging Face Spaces
  • Live App: Launch on Hugging Face

Built a deep learning model that detects pneumonia from chest X-ray images using transfer learning with ResNet50.

Applied data augmentation, early stopping, and fine-tuning techniques to achieve high accuracy on the test set.

Deployed the model via Hugging Face Spaces with a user-friendly interface for real-time predictions.
