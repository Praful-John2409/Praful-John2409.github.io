Trash Classifier

  • Tech Stack: Teachable Machine, TensorFlow.js, Hugging Face Spaces
  • Live App: Launch on Hugging Face

Built a custom image classification model using Google Teachable Machine to classify waste images as biodegradable, recyclable, or landfill.

Integrated the model with TensorFlow.js for browser-side predictions and hosted the interactive app on Hugging Face Spaces.

Designed the project to raise awareness around waste segregation and sustainability practices.
