Motivational Quote Generator

  • Tech Stack: AWS Lambda, S3, Serverless Framework, Node.js, Jimp
  • GitHub URL: Project Link

Created a serverless web app that generates personalized motivational quote images on selected backgrounds (e.g., nature, city).

Used AWS Lambda and Serverless Framework to generate images dynamically and store them in S3, ensuring zero backend infrastructure maintenance.

Enabled local testing via serverless-offline and allowed users to download/share generated images in real time.
